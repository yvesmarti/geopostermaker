# 🗺️ Carto-Art Clone

Éditeur web pour créer des affiches cartographiques artistiques personnalisées.

## ✨ Fonctionnalités

- 🔍 Recherche de lieux via Nominatim (OpenStreetMap)
- 🎨 8 styles artistiques prédéfinis (Minimal, Vintage, Blueprint, etc.)
- 🎨 Palettes de couleurs personnalisables par style
- 🔤 Typographie personnalisable (police, taille, position)
- 🗂️ Toggles de calques (routes, bâtiments, eau, parcs)
- 📤 Export PNG/JPG haute résolution (jusqu'à 300 DPI)
- 📱 Interface responsive avec aperçu en temps réel

## 🚀 Démarrage rapide

### Prérequis
- Node.js 18+ et npm/yarn/pnpm
- Compte GitHub + Vercel (pour le déploiement)

### Installation locale

```bash
# 1. Cloner le repo
git clone https://github.com/votre-user/carto-art-clone.git
cd carto-art-clone/frontend

# 2. Installer les dépendances
npm install

# 3. Lancer en développement
npm run dev
